## 一、AI多模型调用接口（核心）

> 这些接口是AI工具的核心功能，支持接收信息、调用AI模型、传回信息

### 1.1 AI对话接口

**接口说明**：通用AI对话接口，支持多轮对话，可接入不同AI模型

| 属性 | 值                   |
| -- | ------------------- |
| 路径 | `POST /api/ai/chat` |
| 功能 | AI智能对话，支持上下文历史记录    |
| 超时 | 60秒                 |

**请求参数**：

```json
{
  "message": "请解释一下动态规划的核心思想",
  "history": [
    {"role": "user", "content": "你好"},
    {"role": "assistant", "content": "你好！我是AlgoMind学习助手..."}
  ]
}
```

| 参数名     | 类型     | 必填 | 说明                |
| ------- | ------ | -- | ----------------- |
| message | string | 是  | 用户消息内容            |
| history | array  | 否  | 历史对话记录（最多保留最近10轮） |

**响应数据**：

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "response": "动态规划的核心思想是将复杂问题分解为子问题...",
    "timestamp": 1704384000000
  }
}
```

**前端调用**：

```javascript
import api from '@/api'
const response = await api.aiChat('请解释动态规划', history)
```

**后端实现位置**：`AIChatController.java` → `AIService.java`

### 1.5 AIService 多模型调用架构

**后端服务架构**（`AIService.java`）：

```java
@Service
public class AIService {
    // 配置项（支持多模型切换）
    private final AIConfig aiConfig;  // model, temperature, maxTokens, url, key
    
    // 核心调用方法
    public String chat(List<ChatMessage> messages) {
        ChatRequest request = ChatRequest.builder()
            .model(aiConfig.getModel())      // 可配置不同模型
            .messages(messages)
            .temperature(aiConfig.getTemperature())
            .maxTokens(aiConfig.getMaxTokens())
            .stream(false)
            .build();
        return callAIWithTimeout(request);
    }
    
    // 快速失败机制（15秒超时）
    private String callAIWithTimeout(ChatRequest request) {
        // 单次调用，超时自动失败
        // 失败时抛出 AIException，由调用方决定降级策略
    }
    
    // 服务可用性检查（3秒超时）
    public boolean checkAvailability() {
        // 快速检查AI服务是否在线
    }
}
```

**ChatMessage 结构**：

```java
public class ChatMessage {
    private String role;      // system/user/assistant
    private String content;   // 消息内容
    
    public static ChatMessage system(String content);
    public static ChatMessage user(String content);
    public static ChatMessage assistant(String content);
}
```

**多模型支持方式**：

1. 通过 `AIConfig` 配置不同模型的API地址和密钥
2. 通过 `model` 参数指定模型名称
3. 支持流式/非流式响应切换

## 十二、前端API调用封装

### 12.1 API实例配置

**文件位置**：`src/api/index.js`

```javascript
import axios from 'axios'

const api = axios.create({
  baseURL: '/api',
  timeout: 10000,
})

// 请求拦截器：自动注入Token
api.interceptors.request.use((config) => {
  const userRaw = localStorage.getItem('user')
  if (userRaw) {
    const token = JSON.parse(userRaw)?.token
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
  }
  return config
})

// 响应拦截器：统一错误处理
api.interceptors.response.use(
  (response) => response,
  (error) => {
    console.error('API Error:', error)
    return Promise.reject(error)
  }
)
```
